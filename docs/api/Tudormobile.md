## GISLibrary Namespaces
The namespaces available in the *Tudormobile.GISLibrary* library are described below.

### Tudormobile.GISLibrary.dll

```cs
using Tudormobile.GISLibrary;
using Tudormobile.GeoJSON;
```

- [Tudormobile.GISLibrary](Tudormobile.GISLibrary.yml) :link:
    - Root namespace for the library.
    - Essential classes that comprise the api.
    - Extensions for dependency injection.
- [Tudormobile.GeoJSON](Tudormobile.GeoJSON.yml) :link:
    - Facilities for reading and writing GeoJSON data.
    - Facilities for creating and modifying GeoJSON data.
