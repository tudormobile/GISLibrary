# Tudormobile GISLibrary
Tools for accessing and manipulating Geographic information system (GIS) data.  

[Source Code](https://github.com/tudormobile/GISLibrary) :link: | [Documentation](https://tudormobile.github.io/GISLibrary/) :link: | [API documentation](https://tudormobile.github.io/GISLibrary/api/Tudormobile.html) :link:

```bash
dotnet add package Tudormobile.GISLibrary
```
